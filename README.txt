On windows machine, open one command prompt. Change directory to the directory containing all the required files. 

Then run the following commands:
 "javac MainRobot.java AnotherRobot.java Packet.java RobotHandler.java"
 "java MainRobot"


Then open another commandprompt and change directory to directory containing all the require files. Then run this command:
 "java AnotherRobot"

You can then follow the text-based menu to use the program. You can also connect several AnotherRobots by opening multiple new command prompts.
